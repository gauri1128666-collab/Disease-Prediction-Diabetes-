# Disease Prediction System (Diabetes)

A simple machine learning project that predicts whether a patient is likely to have diabetes based on glucose level, BMI, and age.

## Objective
- Predict diseases based on patient medical data
- Assist in early diagnosis
- Support data-driven healthcare decisions

## Problem Statement
Diagnosis in traditional healthcare depends heavily on doctor experience, and early detection of diseases is difficult. This project automates disease prediction using patient data.

## Dataset
| Feature | Description |
|---|---|
| Glucose | Blood glucose level |
| BMI | Body Mass Index |
| Age | Patient age |
| Outcome | Target variable (0 = No Disease, 1 = Disease) |

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn (Logistic Regression)

## Methodology
1. Created a small patient dataset
2. Selected Glucose, BMI, and Age as features
3. Split data into training (80%) and testing (20%) sets
4. Trained a Logistic Regression model
5. Evaluated the model using accuracy and a confusion matrix
6. Predicted the result for a new patient
